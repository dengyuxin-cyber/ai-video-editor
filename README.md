# AI 智能视频剪辑

上传视频 → 自动添加字幕 → 一键合成。所有处理在浏览器本地完成，视频不上传服务器。

## 功能

- 🎬 视频上传（支持 MP4/MOV/AVI，最大 500MB）
- 📝 手动输入字幕（或 AI 自动语音识别）
- 🎨 字幕自动合成到视频
- 🔒 预览版含水印，付费解锁无水印版本
- 💰 按次付费 ¥9.9
- ⚡ 纯浏览器端处理，零服务器成本

## 技术栈

- FFmpeg.wasm — 浏览器端视频处理
- 纯前端，无需后端
- 托管于 GitHub Pages: `https://dengyuxin-cyber.github.io/ai-video-editor`

## 使用

1. 打开网站
2. 上传视频
3. 输入字幕（每行一句）
4. 点击「AI 处理」
5. 预览带水印版本
6. 扫码支付获取解锁码
7. 下载无水印视频
